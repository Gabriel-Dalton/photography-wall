How to use:
  1. Select all images from your /photos folder (Ctrl+A or Cmd+A)
  2. Click "Generate Gallery JSON"
  3. Copy the output and save it as gallery.json in your project root

Note: Make sure to maintain the folder structure when selecting files. The script will preserve relative paths.

CR2 Files: RAW files (.cr2) cannot be processed in the browser. Use the Node.js script (npm run generate) which will automatically convert CR2 files to JPG.
